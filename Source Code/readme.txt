Za instaliranje ovisnosti o projektu (dependency management):
	otvoriti terminal i upisati: pip install -r requirements.txt

Za aplikaciju:
	otvoriti terminal - upisati app.py
	
Za treniranje: 
	otvoriti datoteku training
		ovisno o treniranju slova ili brojeva, upisati - letters_cnn.py ili mnist_cnn.py
		model se sprema pri zavrsetku treniranja u letters.hdf5 / numbers.hdf5 (moguce promijeniti iz koda samo)
